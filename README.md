# Breast Tumor Classification with KNN

## Overview
This project was developed as part of my Bachelor's thesis in Physics and Astrophysics at the University of Florence.

The goal is to classify breast tumors using optical tomography data and machine learning techniques, specifically the K-Nearest Neighbors (KNN) algorithm.

## What I did
- Data preprocessing and feature handling
- Implementation of KNN for classification
- Model training and evaluation
- Analysis of classification performance

## Tools
- Python
- NumPy
- pandas
- scikit-learn
- matplotlib

## Notes
This repository contains a simplified version of the project for demonstration purposes.
